# Login form

1. Download the repo
  `git clone https://github.com/Sa6oZnaka/login-form`
  
2. Edit databaseConfig.js located in the root folder to your MySQL credentials

3. Install the dependencies 
  `npm install`

4. Start the app 
  `npm start`
  
5. Visit https://localhost:3000 in your browser
